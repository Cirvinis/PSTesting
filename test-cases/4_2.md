# Test Case: Scheduled Automated Test Suite Execution

## Test Case Attributes

| Attribute | Value |
| --- | --- |
| Test Case ID | TC-4-2 |
| Title | Verify scheduled execution of Playwright suite |
| Objective | Ensure tests run automatically at a fixed schedule without manual intervention. |
| Requirement | Scheduled run is configured and proof of execution is available. |
| Module | CI/CD / Scheduler |
| Priority | High |
| Severity | Major |
| Preconditions | Repository contains Playwright tests and workflow configuration. |
| Test Data | N/A |
| Environment | GitHub Actions runner (`ubuntu-latest`) |
| Postconditions | HTML report and execution metadata are available as workflow artifacts. |

## Test Steps

| Step | Action | Expected Result |
| --- | --- | --- |
| 1 | Configure scheduled workflow with cron trigger. | Workflow YAML includes `on.schedule` and `workflow_dispatch`. |
| 2 | At scheduled time, workflow starts automatically. | New workflow run appears in Actions history without manual trigger. |
| 3 | Workflow installs dependencies and executes suite. | Playwright tests run and finish with pass/fail status. |
| 4 | Workflow uploads artifacts (HTML report + metadata). | Artifacts are downloadable from the run page. |
| 5 | Open metadata artifact (`scheduler-proof.json`). | File contains run timestamp, event type, and commit SHA. |
